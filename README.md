# Student Management System (Full Stack)

A professional Student Management System built with **Spring Boot**, **MySQL**, and **Bootstrap 5**.

### Features:
- **Full CRUD:** Add, View, Update, and Delete students.
- **Live Search:** Filter students by name as you type.
- **Input Validation:** Prevents empty or invalid data entry.
- **Responsive UI:** Built with Bootstrap 5 for a modern look.

### Tech Stack:
- **Backend:** Java, Spring Boot, Spring Data JPA, Hibernate.
- **Database:** MySQL.
- **Frontend:** HTML5, CSS3 (Bootstrap), JavaScript (Fetch API).
